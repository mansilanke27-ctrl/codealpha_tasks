📊 Exploratory Data Analysis on Top 50 Cryptocurrencies
📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a dataset containing the top 50 cryptocurrencies. The objective of this analysis is to understand market trends, relationships between variables, and identify patterns or anomalies within the cryptocurrency market.

🎯 Objectives

Understand the structure of the dataset

Identify dominant cryptocurrencies based on market capitalization

Analyze price distribution and trading volume

Examine relationships between price and market cap

Detect data issues such as missing values or duplicates

Prepare dataset for further advanced analysis

📂 Dataset Description

The dataset includes the following key attributes:

id – Unique identifier of cryptocurrency

symbol – Trading symbol

name – Cryptocurrency name

market_cap_rank – Rank based on market cap

market_cap_usd – Market capitalization in USD

price_usd – Current price in USD

volume_24h – 24-hour trading volume

high_24h_usd – 24-hour high price

low_24h_usd – 24-hour low price

ath – All-time high price

atl – All-time low price

snapshot_date – Data snapshot date

🧹 Data Cleaning Steps

Converted column names to consistent format

Removed missing values

Removed duplicate records

Verified numeric data types

📊 Exploratory Analysis Performed

Top 10 cryptocurrencies by market capitalization

Price distribution analysis

Market cap distribution analysis

Volume distribution

Correlation heatmap

Scatter plot of price vs market cap

🔎 Key Insights

A small number of cryptocurrencies dominate the market.

Strong positive correlation observed between price and market capitalization.

Market data is highly skewed, with few coins having extremely high values.

Trading volume varies significantly across cryptocurrencies.

🛠 Tools & Technologies Used

Python

Pandas

Matplotlib

Seaborn

Google Colab / Jupyter Notebook

📈 Conclusion

The EDA provided meaningful insights into cryptocurrency market structure and trends. The dataset is cleaned and ready for further statistical analysis or machine learning applications.


📊 Top 50 Cryptocurrency Data Visualization Project
📌 Project Overview

This project analyzes the Top 50 cryptocurrencies using data visualization techniques. The objective is to transform raw crypto market data into meaningful insights using Python libraries.

The project focuses on understanding market trends, price distribution, market capitalization dominance, and relationships between key financial metrics.

🎯 Objectives

Analyze the Top 50 cryptocurrencies.

Compare market capitalization and prices.

Identify dominant coins in the crypto market.

Discover relationships between price, market cap, and trading volume.

Present insights through effective data visualizations.

🛠 Tools & Technologies Used

Python

Pandas

Matplotlib

Seaborn

Google Colab

📂 Dataset

Dataset: Top 50 Cryptocurrency Data

Format: CSV

Key Columns Used:

Name

Symbol

Price (USD)

Market Cap (USD)

Trading Volume

📈 Visualizations Created

Bar Chart – Top 10 Cryptocurrencies by Market Capitalization

Bar Chart – Price Comparison

Pie Chart – Market Share Distribution

Heatmap – Correlation Between Financial Variables

🔍 Key Insights

A small number of cryptocurrencies dominate the overall market capitalization.

High price does not always mean high market cap.

Market cap and trading volume show positive correlation.

The crypto market shows high variation among top-ranked coins.

📊 Conclusion

The project demonstrates how data visualization can convert raw cryptocurrency data into clear and actionable insights. Visual analysis helps in understanding market dominance, trends, and relationships between financial indicators, supporting better investment and analytical decisions.
